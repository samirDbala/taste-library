# 🍽️ Taste Library

A clean, lightweight web app built with **HTML, CSS, and JavaScript** — designed to help you discover delicious recipes from around the world.  
You can search meals, explore ingredients, and view cooking instructions — all powered by a real-time API.

---

## 🚀 Features

- 🔍 **Search recipes instantly** — Find meals by name or keyword  
- 🍲 **Dynamic meal cards** — View recipes with images and categories  
- 📖 **Detailed instructions** — Step-by-step cooking guide  
- 🧂 **Ingredients list** — Automatically extracted with measurements  
- ▶️ **YouTube integration** — Watch recipe videos when available  
- ⚠️ **Error handling** — Handles empty searches & API failures gracefully  
- 📱 **Responsive design** — Works on all screen sizes  

---

## 🧠 Tech Stack

- 🌐 **HTML5** — Structure  
- 🎨 **CSS3** — Styling & layout  
- ⚡ **JavaScript (Vanilla)** — Logic & API handling  
- 🍽️ **TheMealDB API** — Recipe data  
- 🎯 **Font Awesome** — Icons  

---

## ⚙️ How It Works

- User enters a search term  
- App fetches data from TheMealDB API  
- Meals are displayed as cards  
- Clicking a meal shows:
  - Ingredients  
  - Instructions  
  - Video (if available)
